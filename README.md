# 온기척 - 생성형 AI 기반 실시간 보이스 방어 어시스턴트  
1인 가구를 위한 실시간 맥락 기반 망어 멘트 생성 + 자연스러운 남성 음성 재생  
잠재적 침입자·스토킹 상황을 빠르게 억제하는 새로운 AI 안전 서비스

## 프로젝트 개요
- 위험 상황을 텍스트로 입력하면 AI가 맥락에 맞는 방어 멘트를 생성
- 이를 실제 사람 같은 남성 목소리로 재생하는 보이스 방어 시스템

## 핵심 기능
1) 상황 맞춤 방어 멘트 생성 & TTS 즉시 재생
- 사용자가 "누가 문을 두드려요"처럼 현재 상황을 텍스트로 입력
- sLLM이 3~5개의 위기 대응 멘트를 실시간 생성
- 선택한 문장을 자연스러운 남성 음성으로 즉시 재생
- 침입자를 심리적으로 위축시키는 효과

2) 112 긴급 신고버튼
- 대응만으로 해결이 어려운 순간 대비
- 홈 화면에서 1초 만에 즉시 112 신고 전화 연결

3) 자주 쓰는 멘트 '퀵 재생'
- "누구세요?", "문 앞에 놔주세요" 등 자주 쓰는 멘트 저장
- 버튼 하나로 1초 안에 음성 재생

## 활용 데이터
- 감성 음성 데이터셋(AI-Hub)
- 공공 민원 상담 LLM 사전학습 및 Instruction Tuning 데이터(AI-Hub)
- 민간 민원 상담 LLM 사전학습 및 Instruction Tuning 데이터(AI-Hub)

## 사용 기술
1) AI 모델
- 경량 sLLM (Gemma3)
- VITS 기반 오픈소스 TTS
- 모바일 기기에서 구동 가능한 On-Device AI 구조

2) 학습 및 최적화
- PEFT/LoRA 기반 효율적 파인튜닝
- GGUF 4bit/8bit 양자화
